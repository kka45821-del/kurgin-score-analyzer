# Staging Secrets Matrix

## Required server env

| Name | Where | Required | Notes |
|---|---|---:|---|
| KURGIN_API_ENV | `.env.staging` | yes | `staging` |
| KURGIN_API_SECRET | `.env.staging` | yes | generated by `scripts/generate_api_secret.py` |
| KURGIN_ALLOWED_ORIGINS | `.env.staging` | yes | staging platform domain |
| KURGIN_MAX_BATCH_ROWS | `.env.staging` | yes | default 2000 |
| KURGIN_LOG_LEVEL | `.env.staging` | yes | INFO |

## Do not commit

```text
.env.staging
.env.production
real API secrets
supplier Excel files
generated reports
```
